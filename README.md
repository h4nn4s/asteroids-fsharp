# Asteroids in F#

Academic project developed during exchange studies at the University of Copenhagen.

## Description

This project is an implementation of the classic Asteroids game developed in F# as part of a programming and problem-solving course.

The project involved designing and implementing game logic, object interactions, collision handling, and user interaction while applying principles of software design and testing.

## Technologies

- F#
- .NET
- Object-oriented programming
- Functional programming

## Learning Outcomes

Through this project I gained experience in:

- Program design and implementation
- Problem solving and debugging
- Object-oriented and functional programming concepts
- Testing and evaluation of software solutions
- Writing and maintaining structured code

## Context

The project was completed as part of university coursework focused on programming, software design, testing, and software development practices.
